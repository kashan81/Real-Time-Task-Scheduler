Overview

This project is a C++ based simulation of a Real-Time Operating System (RTOS) that demonstrates how a CPU schedules and executes multiple tasks using different scheduling algorithms.

It simulates core operating system concepts such as task management, CPU scheduling, and time-based execution flow using Object-Oriented Programming (OOP).

Objective

The main objective of this project is to understand and implement CPU scheduling mechanisms used in operating systems by simulating:

Priority Scheduling
Round Robin Scheduling
Real-time task execution behavior
CPU time-sharing between multiple processes
Features
Task Management System
Each task has:
Task ID
Priority
Burst Time
Remaining Execution Time
Scheduling Algorithms
Priority Scheduling
Highest priority task executes first
Round Robin Scheduling
Each task gets equal CPU time slice (quantum-based execution)
Gantt Chart Visualization
Displays execution order of tasks over time
Helps visualize CPU scheduling flow
Real-Time Simulation
Time-based execution model
CPU executes one task at a time (simulation of real OS behavior)
